SonicNEAT is a project where a neural network is trained on Sonic the Hedgehog 2 (which is emulated on Genesis Plus GX).

Python 3.8 (no later versions) is required. PyEnv or PyEnv-win is recommended.

To run this, simply clone this repo, install gym-retro (NOT more recent forks such as stable-retro) and add your Sonic 2 ROM as rom.md into retro\data\stable\SonicTheHedgehog2-Genesis\ wherever your Python libraries are stored.

In addition, there is a written report and presentation attached that was presented in my Machine Learning class.
