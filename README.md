SonicNEAT is a project where a neural network is trained on Sonic the Hedgehog 2 (which is emulated on Genesis Plus GX).

To run this, simply add your Sonic 2 ROM as rom.md into retro\data\stable\SonicTheHedgehog2-Genesis\ wherever your Python libraries are stored.

In addition, there is a written report and presentation attached that was presented in my Machine Learning class.

This project was very fun to work on. :D
